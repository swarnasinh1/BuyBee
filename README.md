# 🐝 BuyBee – Smart Shopping Assistant

BuyBee is an AI-powered chatbot that helps users find products based on their needs, preferences, and budget.

## 🚀 Features
- Chat-based shopping assistant
- AI-powered recommendations
- Budget-based suggestions
- Clean pastel UI

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- OpenAI API

## ▶️ How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npx serve  

## ⚠️ Note
Add your OpenAI API key in `.env` file.
